# Static Images

Place your trip cover photos and other images in this directory.

## Recommended Image Specs

- **Trip Cover Photos**: 800x600px or 16:9 aspect ratio
- **Entry Photos**: Maximum 1200px width
- **Format**: JPEG or PNG
- **Optimization**: Compress images before uploading
